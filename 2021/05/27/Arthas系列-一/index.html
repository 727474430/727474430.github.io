Arthas 系列[一] Dashboard

介绍

dashboard 作为 arthas 最基础的功能，为我们提供了丰富的客户化界面，简单直观的展示了我们最为关注的常用信息：

  • thread: 线程信息
  • memory: 内存信息
  • runtime: 运行时信息

使用

启动 arthas

$ java -jar arthas-boot.jar
